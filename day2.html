Q3. What is Hoisting?

Hoisting :Hoisting is a Javascript Mechanism where variables and function declarations are moved to top of there scope before the code execution. Note:Only Declarations are hoisted not the initilization

Q4. What is an temperal dead zone ?

It is the area of a block where a variable is inaccessible until the moment computer initilizes it with a value

TDZ occurs when we declare a variable of Let and Const keyword.
